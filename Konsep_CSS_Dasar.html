BAB 5 CSS SELECTOR

Selector dalam HTML itu adalah pola dalam mencari element-element dalam HTML. Penguasaan tentang "Selector" adalah hal yang paling penting dalam belajar CSS. Ada 16 jenis selector penting yang harus dikuasai.

Sebelum menerapkan CSS ke dalam HTML, hal terpenting yang perlu diperhatikan adalah pada struktur HTML yang teratur dan konsisten. Jika struktur HTML berantakan, maka akan kesulitan dalam penerapan CSS.

Universal Selector

Selector ini menggunakan tanda bintang "*". Selector ini memang simpel, tapi efeknya susah untuk ditimpa jika tidak benar-benar memahami urutan prioritas selector.

Source code di atas menunjukkan cara penggunaan Universal Selector sekaligus efek dalam penerapannya. Di situ terlihat walaupun element selector div mencoba untuk menimpa dan memberikan style pada element div, tetapi hasilnya tetap mengikuti style yang diberikan oleh universal selector.

Oleh karena itu, Universal Selector jarang digunakan karena akan menyulitkan kita dalam mengontrol dan memberikan style pada element HTML yang kita inginkan.

Element Selector

Element selector atau tag selector atau type selector digunakan untuk mencari element HTML berdasarkan nama tag nya.

Element selector sifatnya masih sangat global, sehingga kita tidak bisa mentarget element yang lebih spesifik.

Class Selector

class adalah atribut tag HTML. value/nama dari atribut class pada tag-tag HTML itu boleh sama. Sehingga saat menjadikan nama/value class sebagai selector, itu dapat mentarget satu atau lebih element HTML yang berbeda. ketika dijadikan sebagai selector, maka penulisannya tidak boleh salah (sifatnya case sensitive).

Dalam HTML, satu tag itu bisa memiliki lebih dari satu nama class. Jika semua classnya dipanggil dan diberikan style, maka yang terjadi adalah efek style gabungan.

ID Selector

Id pada HTML juga adalah merupakan atribut, yang mana dalam konsepnya hanya boleh ada satu nama/value id dalam satu halaman HTML.

Untuk menjadikan nama Id sebagai selector itu harus diawali dengan menggunakan tanda pagar "#" sebelum nama Id nya.

Id selector memiliki prioritas yang sangat tinggi, dan dikatakan bahwa sebisa mungkin jangan menggunakan Id selector jika masih bisa menggunakan class selector.
