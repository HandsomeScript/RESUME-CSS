BAB 4 Aturan Dasar Penulisan CSS

Pengertian Selector, Declaration, Property dan Value CSS

Perhatikan contoh kode css di bawah ini h1 disebut sebagai selector. Kemudian setelah selector terdapat declaration yang di dalamnya ada pasangan property dan value.

Selector adalah bagian tertentu dari HTML yang ingin diubah tampilannya. Property adalah nama style yang akan mengubah bagian tertentu dari HTML. Dan value adalah nilai yang dipasangkan pada property tersebut.

Property dan value nya harus dipisahkan dengan tanda titik dua ":"

Kemudian, setiap pasangan property dan value harus diakhiri dengan tanda titik koma.

Case Sensitivity

Untuk penulisan deklarasi, CSS tidak membedakan huruf besar dan kecil (case-insensitive). Tapi untuk penulisan selector CSS bersifat case-sensitive.

Whitespace

Whitespace atau carriage returnsitu adalah karkter yang tidak tampak seperti spasi, tab dan enter. Whitespace ini sangat berpengaruh pada penulisan selector dan deklarasi. Perhatikan baik-baik kode di bawah ini.

Membuat Komentar

CSS hanya mendukung 1 cara dalam penulisan komentar, yaitu menggunakan awalan karakter "/*" dan akhiran "*/".

Semua kode CSS yang berada diantara dua tanda tersbut, tidak akan dibaca oleh web browser. Sehingga ini dapat digunakan untuk mematikan kode dalam proses debugging.

External Style CSS

Cara ini adalah yang sangat disarankan dalam penggunaan style css. Ini dilakukan dengan cara memisahkan CSS pada file tersendiri dengan ekstensi filenya *.css .

Kemudian di dalam file HTML, yaitu di dalam tag <head> , kita panggil file css nya dengan tag <link>. Coba perhatikan kode di bawah ini. Di sini kita diasumsikan sudah memiliki file css nya.

Pada HTML5, kita sudah tidak perlu menambahkan atribut type"text/css", karena di HTML5 ini, CSS sudah merupakan style default yang digunakan.

Kemudian, terlihat pada kode di atas, tag <link> tidak memiliki tag penutup, karena tag tersebut termasuk dalam self closing tag.

Mana yang paling tepat untuk digunakan? Jika website kita memiliki banyak halaman, maka yang paling tepat adalah menggunakan "External File CSS". jika website kita hanya terdiri satu halaman saja, maka "Internal Style CSS" lebih pas untuk digunakan. Untuk "Inline Style CSS" sebaiknya jangan digunakan, karena ini akan menyulitkan kita saat maintenance. Cara ini hanya pas digunakan oleh javaScripts karena JS akan secara otomatis menggenerate style css dengan cara inline style.

CSS Media Type

Ini adalah cara untuk mengatur di media atau perangkat apa file css akan dijalankan. Ada 9 value yang bisa dipilih:

  1. all (default)
  2. screen (sering digunakan)
  3. print (sering digunakan)
  4. aural
  5. braille
  6. handheld
  7. projection
  8. tty
  9. tv

Ada dua cara dalam menggunakan CSS Media Type. Yang sering dipakai adalah menggunakan tag <link>.
